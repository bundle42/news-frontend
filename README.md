# 📈 주가 예측 웹사이트

## 🧩 프로젝트 소개

뉴스 데이터를 기반으로 주가 상승/하락을 예측하는 웹 서비스입니다.

## 🛠 기술 스택

- Frontend: React + Vite + Tailwind CSS
- Backend: Spring Boot / FastAPI
- 기타: Axios, React Router

## 🚀 실행 방법

```bash
npm install
npm run dev
```

## 📌 주요 기능

- 뉴스 기반 주가 예측
- 로그인 / 회원 기능
- 게시판 기능

## 📂 폴더 구조

```plaintext
src/
 ├── components/
 ├── pages/
 ├── context/
 ├── api/
```
